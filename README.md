# Delever Admin Panel

Визуальный редактор контента для сайта Delever.

## Репозиторий
`https://github.com/faxryuk-commits/siteadmin.git`

## Домен
`admin.delever.io` (или отдельный поддомен)

## Основной сайт
Основной сайт находится в отдельном репозитории:
`https://github.com/faxryuk-commits/delever-site.git`

## Запуск
```bash
npm install
npm run dev
```

## Деплой
Проект автоматически деплоится на Vercel при пуше в main ветку.
